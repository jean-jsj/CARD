CARD reference results — can a good forecaster recover causal demand?

Four reference models, dev seed 1. Each is the cell's own demand family fit from the public files; they differ only in which inputs they use — the released cost instruments, the product text, both, or neither. Pick a demand family; every chart below shows that family.

own-price bias — signed error on the price-changed product's demand response (WMPE) · 0 = unbiased, sign = direction
substitution error — error on competitor products' demand change (WAPE) · 1.0 = predicting no change · lower is better
forecast error — held-out sales error (revenue-weighted WMAPE) · lower is better

1 · Forecast skill does not buy causal recovery

Top: the endogeneity-on arena cell. Bottom: the control cell — identical draws, but discounts no longer respond to the hidden shock. If fitting observed demand were enough, points would hug the zero line in both panels; the vertical movement between the two panels is the bias the endogeneity coupling induces (the same movement panel 2 shows as connectors).

2 · What flipping the endogeneity switch does to own-price bias

The on/off pair draws byte-identical shocks, costs, and promotion timing — the toggle changes one coupling constant. The connector is the bias that coupling induces.

3 · How much substitution structure is recovered

The substitution error (WAPE) scores the predicted demand change on competitor products. Predicting "no change for competitors" scores exactly 1.0 — the dashed line. The gap between a bar and that line is the substitution structure the model actually recovered; 0 would be a perfect substitution map.
